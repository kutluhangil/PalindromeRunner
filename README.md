# Palindrome Runner

Runner tipi bir oyun: ilk yarıda koş ve tıkla, ikinci yarıda aynı seviyede tıklamaların zaman-aynası olmak zorunda. Başarı için kendi input dizini palindroma çevir.

## Çalıştırma

```bash
npm install
npm run dev        # http://localhost:3000
npm run build
npm run preview
npm test
npm run typecheck
```

## Belgeler

- `CLAUDE.md` — Claude Code için kalıcı proje bağlamı ve mimari kurallar.
- `BUILD_PROMPTS.md` — Fazlara ayrılmış iş emri deposu (sırayla uygulanır).
